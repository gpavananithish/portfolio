# 🚀 GPN Personal Portfolio

Welcome to my personal portfolio website! This project is a showcase of my journey as an aspiring Software Developer, highlighting my skills, projects, and educational background with a premium, responsive design.

🔗 **Live Link:** [gpavananithish.vercel.app](https://gpavananithish.vercel.app/)


## ✨ Features

- **Modern UI/UX**: Built with a sleek, dark-themed aesthetic featuring glassmorphism and smooth animations.
- **Micro-interactions**: Interactive terminal, custom cursor, and dynamic marquee for technical skills.
- **Responsive Design**: Optimized for everything from high-resolution monitors to mobile devices.
- **Dynamic Projects**: Detailed showcase of key projects with integrated tech stack badges.
- **Performance**: Powered by Vite and React for lightning-fast load times.

## 🛠️ Tech Stack

### Frontend
- **React 19**: Modern UI framework.
- **GSAP & Framer Motion**: Advanced animations and scroll triggers.
- **Bootstrap 5**: Responsive layout grid.
- **React Icons**: Comprehensive icon library.

### Backend & Database
- **Django**: Robust Python web framework.
- **SQLite / MySQL**: Reliable data management.

### Deployment & Tools
- **Vite**: Next-generation frontend tooling.
- **EmailJS**: Direct contact form integration.

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gpavananithish/My_Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd My_Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
